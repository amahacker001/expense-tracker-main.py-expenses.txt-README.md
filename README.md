

💰 Expense Tracker Web App

🚀 Features

- Add expenses
- Display expense list instantly
- Clean and simple interface

🛠️ Built With

- HTML
- CSS
- JavaScript

📂 Project Structure

- index.html → Main page
- style.css → Design
- script.js → Functionality

▶️ How to Use

1. Enter expense name
2. Enter amount
3. Click "Add"
4. View your expenses below
